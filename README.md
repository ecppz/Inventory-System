# Node CRUD App - Sistema de inventarios

Sistema de gestión de inventario de productos desarrollado con **Node.js, Express y MongoDB**. Incluye autenticación de usuarios, manejo de sesiones y un CRUD completo.

---

## 🚀 Funcionalidades principales

### 👤 Autenticación de usuarios
- Registro e inicio de sesión
- Gestión de sesión con express-session

### 🛒 Gestión de inventario (CRUD de productos)
- Crear, editar y eliminar productos
- Listado dinámico desde MongoDB
- Control de stock y precios

---

## 🧠 Caracteristicas del proyecto

- Sistema completo de gestión de inventario
- CRUD funcional con base de datos real (MongoDB)
- Manejo de sesiones en Express
- Protección de rutas privadas
- Manejo de archivos en servidor (upload system)

---

## 🧩 Tecnologías usadas

- ⚙️ Node.js
- 🚀 Express.js
- 🗄️ MongoDB + Mongoose
- 📦 Multer
- 🔐 express-session
- 🧾 EJS
